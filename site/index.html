Get started in two steps

Add the marketplace to Claude Code once, then install any plugin.

01 — ADD MARKETPLACE

Register the marketplace

Tell Claude Code where to find this catalog. You only need to do this once.

$/plugin marketplace add Seretos/agent-marketplace
02 — INSTALL A PLUGIN

Install any plugin

Pick a plugin from the catalog below and run its install command — each card has its own copy button.

$/plugin install <name>@agent-marketplace
